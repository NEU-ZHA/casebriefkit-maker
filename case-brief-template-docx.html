How to use the DOCX file

  1. Download the DOCX file and save a clean copy.
  2. Duplicate the file for each assigned case.
  3. Fill in facts, issue, rule, analysis, holding, reasoning, and class notes.
  4. After class, update the brief with professor corrections or exam cues.

Editable but not overbuilt

The goal is speed. The template avoids decorative formatting, locked fields, and complicated macros because law students usually need a fast note structure more than a designed document.

DOCX vs PDF

Choose DOCX if you want to type, revise after class, or keep a folder of briefs for exam review. Choose PDF if you want to print before reading and write by hand.
